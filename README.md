# Google Search UI Clone

A front-end recreation of the Google Search homepage using **HTML** and **CSS**. This static clone emulates the layout and responsiveness of the official Google interface as of 2020.

---

## Features

- Logo, search bar, and button layout mimicking Google
- Responsive design using media queries
- Fixed footer with left and right navigation links
- Sign-in button with hover effect
- Minimalist, mobile-first layout

---

## Project Structure

| File             | Description                          |
|------------------|--------------------------------------|
| `google search.html` | Main HTML file for Google UI         |
| `google.css`         | Primary stylesheet for layout         |
| `stylesheet.css`     | Additional styling (if separated)     |
| `google logo.png`    | Google logo used for display          |
| `text.html`          | Supplementary content (optional)      |

---

## How to Use

1. Open `google search.html` in a web browser.
2. View and interact with the search interface.
3. Resize the window to observe responsiveness.
4. Customize or integrate real search logic for backend functionality.
